## 	:toolbox: Getting Started

<!-- Installation -->
### :gear: Installation

Install my-project with npm

```bash
  npm install
  cd MotoHack_GP
```

<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/xcapde/MotoHack_GP
```

Go to the project directory

```bash
  cd MotoHack_GP
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
