## 	:toolbox: Getting Started

<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/xcapde/MotoHack_GP-UI.git  
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
